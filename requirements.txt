# Job Manager — system & runtime requirements

System
  macOS or Linux
  Node.js >= 20.x          (developed against v24.14.1)
  npm    >= 10
  Native build tools for better-sqlite3:
    macOS:  Xcode Command Line Tools   (xcode-select --install)
    Linux:  build-essential, python3

Runtime dependencies (installed by `npm install`)
  better-sqlite3   ^12.9.0    local SQLite database
  express          ^4.21.2    API server
  gray-matter      ^4.0.3     Obsidian markdown frontmatter parser
  react            ^19.0.0    UI framework
  react-dom        ^19.0.0
  vite             ^6.2.0     dev server / bundler
  tailwindcss      ^4.1.14    styling
  motion           ^12.23.24  navigation animations
  lucide-react     ^0.546.0   icon set
  date-fns         ^4.1.0     date formatting
  clsx             ^2.1.1     className utility
  tailwind-merge   ^3.5.0     className merge

Dev dependencies
  typescript       ~5.8.2
  tsx              ^4.21.0    runs server.ts directly (no build step in dev)
  @types/express, @types/node, @types/better-sqlite3
  autoprefixer     ^10.4.21

Quick start
  npm install
  npm run dev      # starts on http://localhost:3000
  npm run lint     # tsc --noEmit
  npm run build    # vite production bundle

Optional environment variables (see .env.example)
  DB_PATH          path to local SQLite file (defaults to ./jobs.sqlite)
